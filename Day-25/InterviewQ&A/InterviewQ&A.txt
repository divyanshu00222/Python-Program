Q1. What is a linked list?

A data structure where elements (nodes) are connected using pointers.

Q2. Difference between array and linked list?
Arrays	Linked List
Fixed size	Dynamic
Random access	Sequential access
Insertion slow	Insertion fast
Q3. Time complexity of insert at end?

O(n)

Q4. Can linked list be used for stacks and queues?

Yes. Very common.

Q5. Why is linked list important?

Used in memory management, queues, graphs, OS, and interviews.
