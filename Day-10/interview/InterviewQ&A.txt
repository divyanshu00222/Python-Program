✅ Q1. Difference between CSV & JSON?
CSV	JSON
Table format	Object format
Simpler	More structured
No nested data	Supports nested data
✅ Q2. What is serialization?

Converting Python objects → JSON / file format.

✅ Q3. What is deserialization?

Converting JSON/file → Python object.

✅ Q4. Why is CSV used?

Lightweight, fast, easy for spreadsheets & databases.

✅ Q5. Can JSON store lists & dictionaries?

✅ Yes, JSON supports nested lists, dictionaries, arrays.
